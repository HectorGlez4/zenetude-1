Qu'est-ce que Zenetude ?

Zenetude est un projet de Learning Management System (LMS) purement Web : tous les accès se font via un navigateur Web standard, à travers Internet (connexions sécurisées). Il a pour vocation initiale d’être essentiellement orienté étudiants (avec un accès exclusif, ou privilégié consacré à ces derniers).
